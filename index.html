Visual Dashboard of Litter Collection

This dashboard visualizes litter collection data from Prince George's County, Maryland. It tracks the collection of trash, debris, and tires by residents, community organizations, and businesses during individual and group clean-up events across different council districts.

Data Source: Prince George's County LitterTRAK Dataset

Note: Due to API limitations, this visualization shows 1,000 most recent records from the original dataset of 11,300 entries.
